区块链发展

币 链 网
BTC, LTC,达世币, Zcash 侧链，以太坊，超级账本 DHG

DHG(Distributed Hybrid Global)是一个区块链技术综合应用平台，融合区块链发展至今的众多前沿技术，例如UTXO、侧链、SPV验证、混合挖矿以及智能合约等，并不断探索完善区块链底层协议，顺应区块链发展潮流，在虚拟币和区块链发展基础上，定义下一代区块链网络协议，同时也是物联网的基础协议。

应用场景

= 数字资产 =

资产数字化后可量化，可自由流通，买卖，抵押，因而将会催生巨大价值。DHG将提供多种资产数字化方法，可以通过侧链开通一个应用，并同时发行一种数字资产，也可以通过智能合约构造资产。

= 数字身份 =

DHG区块网可用于数字身份，食品，药品，工艺品，文物等领域，以证明数据及产品所有权问题。区块链在产品供应链溯源领域同样具备天然的优势，不可篡改的区块链记录可清晰的标记产品流通轨迹。

= 社会治理 =

传统领域，身份认证，健康管理，公证，司法仲裁，投票，借贷等，使用中心化系统都会存在造假问题。区块链天然的具备公开透明，不可伪造的属性。DHG的侧链机制将可以对数据进行分级处理。

= 去中心化应用(DAPP) =

在去中心化运算及去中心化数据存储的基础之上，可发展去中心化应用，DAPP也是物联网的发展基础。DHG最终可提供两种搭建DAPP的方式，即通过侧链的方式实现，或者通过智能合约的方式实现。
